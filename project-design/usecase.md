- #### 유스케이스 구성요소
  - 액터, 시스템, 유스케이스, 관계

    
- #### 관계 설명
  - 연관관계 : 액터와 유스케이스 간의 상호작용 있음.
  - 포함관계 : 로그인 유스케이스가 반드시 실행 되어야만 게시판 관리가 가능
  - 확장관계 : 게시판 목록조회 유스케이스가 부가적으로 게시판 상세조회 기능 추가 가능
  - 일반화관계 : 게시판 관리가 추상적인 유스케이스고 게시판 등록, 수정, 삭제가 구체적인 유스케이스
